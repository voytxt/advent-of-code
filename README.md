# Advent of Code

## How to run

1. Paste the input into `io/input.txt`
2. Run `npm start [year] [day]-[part]` or `npm run watch [year] [day]-[part]` (for example `2022 3-1` would be year 2022, day 3, part 1)
3. Get the output from `io/output.txt`
