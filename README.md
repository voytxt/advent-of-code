# Advent of Code

## How to run

1. Paste the input into `io/input.txt`
2. Run `deno task run [year] [day]-[part]` or `deno task watch [year] [day]-[part]` (e.g. `2022 3-1` would be year 2022, day 3, part 1)
3. Get the output from `io/output.txt`
